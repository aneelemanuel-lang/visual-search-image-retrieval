# Visual Search and Image Retrieval

## Overview
This project implements a content-based image retrieval system for visual search over an image collection. It compares colour, spatial, texture and PCA-based descriptors using multiple distance metrics.

## Methods
- Global Colour Histogram
- 3×3 Spatial Colour Histogram
- Edge Orientation Histogram / texture descriptor
- PCA-based descriptors
- Late fusion of colour and texture features

## Evaluation
- Precision–Recall curves
- Mean Average Precision (mAP)
- Confusion matrix
- Qualitative retrieval visualisation
